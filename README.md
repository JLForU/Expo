# SD_Expo
Repositorio para trabajo expositivo de MQTT

## Ejecución
0. `sudo systemctl status mosquitto`
1. Copiar `script.sh` a `<directorio>`.
2. `cd <directorio>`
3. `sudo chmod +x script.sh`
4. `./script.sh`
