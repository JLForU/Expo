# SD_Expo
Repositorio para trabajo expositivo de MQTT

## Ejecución
0. `systemctl status mosquitto`
1. `cd demo`
2. `python3 subscriptor.py`
3. Abrir otra terminal.
4. `python3 publicador.py`
